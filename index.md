## GSCA Overview

* Help users to more efficiently explore Publicly available gene Expression Data (PED).

* R/Bioconductor software package with four Affymetrix data packages.

* GUI built using R/shiny.

---

## Challenges

* Massive amounts of PED, but not fully utilized

* Microarray data generated by lab A studies cell type B in pathway C, but also contain information about pathway D.

* re-use of PED impeded by lack of software tools and tedious data analysis procedures

<center><img src="fig/Fig1.png" width="70%"></center>

---

## GSCA

* Built upon 30000+ pre-normalized and expert-annotated human and mouse gene expression samples with 1000+ biological contexts.

* Explore PED conveniently with the GUI

* Link highly-customizable genes expression patterns to biological contexts
